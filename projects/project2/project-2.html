Supplies

This is a tutorial for a beginner-friendly friendship bracelet pattern, the striped friendship bracelet. First, you will need embroidery thread. You can use as many, and how many colors you want, but for this example, I will be using 4 colors. You will also need tape, and a pair of scissors.
